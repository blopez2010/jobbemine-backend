# JobBeMine-backend

Backend repository to serve bios, opportunities and search engines for job search opportunities

## Requirements

1. IDE: VS Code.
2. NodeJs Latest.
3. Git.

## Installation

Open the terminal and run the following:

1. Run `npm install yarn -g`.
2. Run `yarn`
3. Run `F5` for debugging. (VS Code *ONLY*)
4. Run `yarn start`.
